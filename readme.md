# SmartClinic

Sistema web desenvolvido para simular a plataforma de uma clínica médica moderna.

## 📌 Sobre o projeto

O projeto foi desenvolvido com foco em:

- Experiência do usuário
- Interface moderna
- Organização de código
- Responsividade
- Estrutura profissional de front-end

A aplicação simula o site institucional de uma clínica médica, contendo áreas como:

- Página inicial
- Especialidades médicas
- Avaliações
- Planos atendidos
- Localização
- Contato

---

## 🚀 Tecnologias utilizadas
